Welcome to MicroPet, a delightful Tomagotchi clone designed specifically for the Raspberry Pi Pico using MicroPython. This project leverages the power of the Pico's dual M0 cores and implements a double buffer system for efficient graphics processing, ensuring a smooth and engaging virtual pet experience.
Features:

    Raspberry Pi Pico Compatibility: Tailored for the Raspberry Pi Pico microcontroller, providing a cost-effective and versatile platform.
    MicroPython Implementation: Written in MicroPython, making it accessible and easy to modify for Python enthusiasts.
    Dual M0 Core Utilization: Takes full advantage of the Pico's dual-core architecture for enhanced performance and responsiveness.
    Efficient Graphics with Double Buffer: Optimized graphics processing using a double buffer system to minimize flickering and deliver a visually pleasing user interface.
    Interactive and Engaging: Experience the joy of raising a virtual pet with interactive features, feeding, playing, and more.

Getting Started:

    Clone this repository to your Raspberry Pi Pico.
    Install MicroPython on your Pico if not already done.
    Upload the MicroPet code to the Pico using your preferred method.
    Power on the Pico and enjoy the virtual pet experience!

Dependencies:

    MicroPython for Raspberry Pi Pico
